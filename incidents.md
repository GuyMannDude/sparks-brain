## How to use this file

1. When something breaks and you spend more than 10 minutes finding the cause, log it here.
2. Always log the **symptom** — the search-friendly description of what looked wrong. That's how future-you finds this entry.
3. Don't log obvious bugs (typo, missing import). Log the ones where the cause was non-obvious.
4. Keep entries short — symptom + cause + fix is usually three sentences.
