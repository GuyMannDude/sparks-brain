## How to use this file

1. When you make a non-obvious choice, log it here. Even small ones — "use library X over Y because of Z."
2. When you reverse a decision, add a new entry on top and add `**Superseded by:** [link or date]` to the old one. Don't delete the old reasoning.
3. When you read code that confuses you, check here. If the rationale isn't logged, that's a gap worth filling.
